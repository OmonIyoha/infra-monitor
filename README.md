\# Infrastructure Monitoring Tool



Python-based infrastructure monitoring tool for tracking CPU, memory, and disk usage within local environments.



\## Features



\- CPU monitoring

\- Memory monitoring

\- Disk monitoring

\- Log generation

\- Automated monitoring checks

\- Alert detection for high CPU usage



\## Technologies Used



\- Python

\- psutil

\- Git

\- GitHub
